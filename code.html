                                document.getElementById('alert').classList.add('invisible')

                            function checkPalindrome(userInput){
                                let userString = userInput.toLowerCase().replace(/[^a-zA-Z]/g, ''); 
                                let splitString = userString.split("")  
                                let reversedArray = splitString.reverse();   
                                let joinArray = reversedArray.join("")  
                                return joinArray
                            }

                            function displayResults(userInput, reversedString){  
                                let fixedUserString = userInput.toLowerCase().replace(/[^a-zA-Z]/g, '')
                                let messageDisplay = document.getElementById('message') 
                                let headingMessage = document.getElementById('the-heading') 

                                messageDisplay.innerHTML = ''; 
                                headingMessage.innerHTML = '';

                                document.getElementById('alert').classList.remove('invisible')  

                                if(fixedUserString !== reversedString){
                                    headingMessage.innerHTML = `${userInput} is not a palindrome`
                                    messageDisplay.innerHTML = `Your phrase reversed is: ${reversedString}`
                                }else{
                                    headingMessage.innerHTML = `Well done! You entered a palindrome`
                                    messageDisplay.innerHTML = `Your phrase reversed is: ${reversedString}`
                                }
                            }
getText()

The function getText() gets the user string and passes it as an argument to other functions inside of the program. This function also adds a class of invisible to the alert div in order to hide it when the button in the form has not been clicked.


checkPalindrom()

The checkPalindrome function takes in userInput as an arquement and performs some operations on it. The function begins by converting the user input to all lower case and removing everything that is not upper case or lower case a-Z. I then split the string into the array using the split method. After using the split method I then user the reverse method to reverse the array. Finally I use the join method to convert the array back into a string that is reversed from the original.


displayResults()

The displayResults function takes in userInput and the reversedString as arguements. In order to determine if the reversedString arguement is a palindrome I have to remove all non letter characters from userInput and conmvert to lowerCase. I then get the message and the-heading elements from the alert div. Using an if and else statement I check to see if the userInput is equal to reversedString and then I edit the html in the alert div based on that
